# Lesson 18: Looping Lists — Teaching Guide

## Concept Introduction
"for item in list: lets you visit each item. enumerate() adds a counter. zip() walks two lists side by side."

## Prediction Question
"What does `for i, x in enumerate(['a','b','c'], 1)` give for the first iteration?"

## Vocabulary Introduced
- **enumerate()** — loop with index + item
- **zip()** — loop two lists in parallel
- **.join()** — combine list of strings into one string
