# Project 1: Mad Libs — Hints

## Hint 1: Getting started
Write the story first with hardcoded words, then replace them with variables.

## Hint 2: Asking for words
```python
noun = input("Give me a noun: ")
adjective = input("Give me an adjective: ")
```

## Hint 3: Building the story
```python
print(f"The {adjective} {noun} went to the store.")
```

## Hint 4: Multiple lines
You can use multiple print() calls or use triple quotes:
```python
print(f"""
One day, a {adjective} {noun} decided to {verb}.
It went to the {place} and ate a {food}.
""")
```
