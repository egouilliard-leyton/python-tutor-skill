# Lesson 21: Defining Functions — Teaching Guide

## Concept Introduction
"A function is a reusable block of code with a name. Write it once, use it anywhere."

Analogy: "A recipe in a cookbook — write it once, follow it every time you want that dish."

## Prediction Question
"If I define `def add(a, b): return a + b` and then write `result = add(3, 4)`, what is result?"

## Common Mistakes
### 1. Using print instead of return
→ "print shows text on screen. return sends the value BACK so you can use it."

### 2. Forgetting the colon
→ "def greet(name) needs a colon at the end: def greet(name):"

## Vocabulary Introduced
- **function** — reusable named block of code
- **def** — keyword to define a function
- **parameter** — variable in the function definition
- **return** — send a value back from the function
